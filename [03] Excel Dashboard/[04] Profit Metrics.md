### Profit Metrics

- Total Profit
```
=SUM(Dataset[profit])
```
Custom Formatting -> $0.00,,"M"

- Average Profit per Order
```
=M3/B5 [= Total Profit/ Total Orders]
```

- Profit Margin
```
=M3/C3 [= Total Profit/ Total Revenue]
```

- Sales Growth Rate
```
=(Sales in 2021−Sales in 2016) / Sales in 2016
```
Custom Formatting -> %

- Total Units Sold
```
=SUM(Dataset[quantity])
```
